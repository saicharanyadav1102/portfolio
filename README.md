## ✨ Features

- **🌓 Dynamic Light & Dark Theme Toggle**: Switch seamlessly between a crisp Light Mode and high-contrast Dark Mode with persistent memory using `localStorage`.
- **⚡ Responsive & Mobile-First Architecture**: Built using Tailwind CSS utility classes to ensure flawless rendering across smartphones, tablets, and desktop screens.
- **🖼️ Real-Time Profile Photo Preview**: Upload and preview your profile picture instantly on the client side using the browser's native `FileReader` API.
- **🎭 Scroll-Triggered Micro-Animations**: Smooth section entrances and staggered timeline animations powered by native `IntersectionObserver`.
- **📜 Clean Modular Codebase**: Organized into clean structural (`index.html`), styling (`style.css`), and logical (`script.js`) layers.

## 🚀 Getting Started

### Prerequisites
No Node.js or heavy build tools are required! This website runs directly in any modern web browser.

### Running Locally

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio/portfolio11
   ```

2. **Open the Website**
   - Simply double-click `index.html` to launch it in your browser (`file:///path/to/portfolio11/index.html`).
   - *Optional:* If using VS Code, use the **Live Server** extension for real-time reloading during development.

## ⚙️ Customization Guide

- **Profile Image**: Replace `sai.jpg` with your own photo, or use the live image upload feature directly on the page.
- **Content & Bio**: Edit text inside `index.html` to customize sections, links, and personal accomplishments.
- **Theme Palette**: Default colors use clean `bg-white text-gray-900` for Light Mode and `dark:bg-black dark:text-gray-200` for Dark Mode. Modify utility classes in `index.html` to adjust branding.
