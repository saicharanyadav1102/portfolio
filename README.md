# KV Saicharan Yadav — Developer [Portfolio](https://portfolio-saicharan-23.vercel.app/)

A personal developer portfolio built with **HTML5**, **Tailwind CSS**, and **Vanilla JavaScript**. Designed with a clean editorial aesthetic, responsive layout, and interactive components.



---

##  Overview

- **Modern & Responsive:** Fully responsive layout built with utility-first Tailwind CSS.
- **Dark / Light Theme:** Switch between clean light mode and high-contrast dark mode with automatic `localStorage` saving.
- **Interactive Components:** Features mouse spotlight effects, a 3D flip quote card, and animated timelines.
- **Live Contact Form:** Real-time email delivery using [Web3Forms](https://web3forms.com/) with a clean confirmation popup.

---

##  Built With

- **HTML5** (Semantic structure & sections)
- **Tailwind CSS** (Styling & layout)
- **Vanilla JavaScript** (Interactivity, animations, & theme toggle)

---

##  Running Locally

No installation or build tools required. To run locally:

1. Clone or download this repository:
   ```bash
   git clone https://github.com/saicharanyadav1102/portfolio11.git
   ```
2. Open `index.html` directly inside your web browser (`double-click the file`).
